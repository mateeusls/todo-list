# Todo List

A simple and efficient Todo List application built with React.

## Features

- ⚠️ Add new tasks
- ⚠️ Mark tasks as complete
- ⚠️ Delete tasks
- ⚠️ Filter tasks (all, active, completed)
- ⚠️ Persistent storage

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

The application will run on `http://localhost:3000`

## Build

```bash
npm run build
```

## Technologies

- React
- Vite
- Typescript
- Tailwindcss
